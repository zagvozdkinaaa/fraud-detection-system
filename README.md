# Banking Fraud Detection System

End-to-end anti-fraud system based on:

- XGBoost
- Isolation Forest
- MobileNetV2

## Project Structure

1. Fraud Classification (XGBoost)
2. Anomaly Detection (Isolation Forest)
3. Face Verification (MobileNetV2)

## Results

Best model: XGBoost

- Recall: 85%
- ROC-AUC: 0.986
- PR-AUC: 0.485

Financial Transactions Dataset:
https://www.kaggle.com/datasets/computingvictor/transactions-fraud-datasets
